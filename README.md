- Quinta a sexta: 4 horas dedicado a uma tecnologia

# Áreas da tecnologia para especializar

- **Java: é uma linguagem de programação orientada a objetos de alto nível, conhecida por sua portabilidade, segurança e robustez. É amplamente utilizada em aplicações empresariais, desenvolvimento de jogos e sistemas embarcados.**

- **JavaScript: é uma linguagem de programação de alto nível, interpretada e multiplataforma. É usada principalmente para criar efeitos interativos em páginas web e desenvolvimento de aplicativos front-end.**

- **Python: é uma linguagem de programação de alto nível, interpretada e de propósito geral, utilizada em áreas como ciência de dados, inteligência artificial, desenvolvimento web, entre outras.**

- **HTML: é a linguagem de marcação utilizada para estruturar conteúdo em páginas web.**

- **CSS: é uma linguagem de estilo que define a apresentação de conteúdo em páginas web, incluindo layout, cores, fontes e animações.**

- **Spring: é um framework Java para desenvolvimento de aplicações web e empresariais, que oferece diversos recursos para agilizar o desenvolvimento, como injeção de dependência e tratamento de exceções.**

- **React: é uma biblioteca JavaScript para construção de interfaces de usuário, utilizada para criar componentes reutilizáveis e de fácil manutenção.**

- **Firebase: é uma plataforma do Google para desenvolvimento de aplicativos móveis e web, que fornece diversos recursos como autenticação de usuários, armazenamento em nuvem, banco de dados em tempo real, entre outros.**

- **MongoDB: é um banco de dados NoSQL orientado a documentos, que oferece alta escalabilidade, flexibilidade e performance para aplicações modernas.**

- **MySQL: é um banco de dados relacional de código aberto, amplamente utilizado em aplicações web e empresariais.**

- **Cassandra: é um banco de dados distribuído NoSQL, projetado para lidar com grande volume de dados e alta disponibilidade.**

- **Figma: é uma ferramenta de design de interfaces de usuário baseada na web, que permite a criação de layouts, protótipos interativos e colaboração em tempo real.**

- **Arduino: é uma plataforma de prototipagem eletrônica de código aberto, que permite criar projetos interativos de forma simples e acessível.**

- **Linux: é um sistema operacional de código aberto, conhecido por sua estabilidade, segurança e flexibilidade.**

- **Azure: é uma plataforma de computação em nuvem da Microsoft, que oferece diversos serviços para desenvolvimento e hospedagem de aplicativos.**

- **AWS: é uma plataforma de computação em nuvem da Amazon, que oferece diversos serviços para desenvolvimento e hospedagem de aplicativos.**

- **Heroku: é uma plataforma de hospedagem em nuvem, que permite implantar e gerenciar aplicativos de forma fácil e rápida.**

- **Git é um sistema de controle de versão distribuído de código aberto, que permite aos desenvolvedores rastrear mudanças em seu código-fonte ao longo do tempo e colaborar com outras pessoas em projetos de software.**
- **GitHub é uma plataforma de hospedagem de código-fonte baseada em nuvem que permite aos desenvolvedores hospedar, gerenciar e colaborar em projetos de software.**

## *Projetos para portfólio 2023*
- **Desenvolvimento de um sistema baseado em Machine Learning para auxiliar estudantes do ensino médio a escolherem suas futuras profissões. O sistema coletará dados de profissionais e utilizará algoritmos de aprendizado de máquina para fornecer recomendações personalizadas aos estudantes. Além disso, será criado o website meufuturoprofissional.com.br, onde os alunos poderão fornecer informações sobre suas habilidades, interesses e preferências, permitindo que o sistema faça uma análise mais precisa e eficaz. Dessa forma, os estudantes terão acesso a informações valiosas sobre possíveis carreiras e poderão tomar decisões mais informadas e conscientes sobre seu futuro profissional.**

- **Desenvolvimento de um software para armazenar informações dos alunos que entram ou saem da escola.**

- **Desenvolvimento de um software para criação automática de provas com base no conteúdo abordado em determinado período, visando agilizar e facilitar o processo de elaboração de avaliações pelos professores.**

## *Estudos para a criação da empresa 2023*

- Estudo do mercado e concorrência
- Definição do público-alvo e segmento de atuação
- Elaboração do plano de negócios
- Análise de viabilidade financeira
- Escolha do tipo de empresa (MEI, EIRELI, LTDA,etc.) e regime tributário (Simples Nacional, Lucro Presumido, etc.)
- Registro da empresa nos órgãos competentes (Junta Comercial, Receita Federal, Prefeitura, etc.)
-Obtenção de alvarás e licenças necessárias para o funcionamento do negócio
- Contratação de funcionários, se necessário
- Definição de estratégias de marketing e comunicação
- Investimento em tecnologia e infraestrutura adequadas para o negócio
- Elaboração de contratos, termos de serviço e políticas de privacidade, se aplicável
- Estudo de regulamentações e normas específicas para o setor de atuação
- Estabelecimento de parcerias estratégicas e networking com outros profissionais e empresas do mesmo ramo.
